JS1k Piano

This is an entry for the JS1k competition, to produce something interesting in 1k (or less) of JavaScript. This entry is a simple music generator, controlled by the keyboard in a manner similar to a piano.

Important: This application uses technologies which are not (as of August 2010) supported by all web browsers. Please see the compatibility table to see if your browser is one which provides the necessary functionality.

How to use this

The QWERTY and number rows of your keyboard play the notes. Keys Q,W,E,R,T,Y,U and I play the white notes (starting from C), keys 2,3,5,6 and 7 play the black notes.

(Apologies to international users with keyboard layouts such as AZERTY or QWERTZ.)

Please be patient when you first start the application, as it has to create the sound waves for all the musical notes. This can take several seconds, depending on the speed of your browser, your computer, etc.

Browser compatibility

This application requires that your browser supports the <audio> element introduced in HTML5. Further, it requires that the browser support the WAV audio format. and data URIs. (See the technical details section about what these things actually mean.)

Obviously you will need to enable JavaScript in your browser. If you don't know what this means, or how to do it, don't worry - it will almost certainly be enabled by default.

The table below only lists browser/operating system configurations which I have personally tested, and is by no means a definitive or authoritative list.
